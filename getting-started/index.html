    Understanding Prompt Structure

    Each prompt in our collection follows a consistent structure:

    # [Purpose]

## Context
[Background information and when to use this prompt]

## Variables
- {variable1}: Description
- {variable2}: Description

## Prompt Template
[The actual prompt template with variables]

## Example Usage
[Practical example with variables filled in]

## Best Practices
[Tips for getting the best results]

    How to Use a Prompt

    1. Choose the Right Category
    2. Browse through our categories to find the most relevant prompt

    3. Use the search function to find specific keywords

    4. Customize the Prompt

    5. Replace variables with your specific information
    6. Keep the overall structure intact

    7. Test and iterate as needed

    8. Best Practices

    9. Be specific with your requirements
    10. Provide relevant context
    11. Use clear and concise language
    12. Test the prompt with different variations

    10. Contribute your own prompts
    11. Help improve existing ones
